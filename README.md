# 贝塔驿站微信小程序

贝塔驿站是面向校园生活场景的小程序项目，主要覆盖集市信息、树洞交流、内容发布、搜索、评论互动、个人中心和轻量工具等功能。项目采用微信小程序原生能力开发，并结合 Vant Weapp、WeUI 扩展库和自定义组件组织页面体验。

本文档只描述项目结构、功能范围、开发流程和安全注意事项，不包含 AppID、密钥、接口域名、个人联系方式、真实用户数据或代码片段。

## 项目定位

贝塔驿站的核心目标是提供一个校园内高频信息流转入口，让用户可以快速浏览、筛选、发布和管理校园内容。当前小程序包含三个主要入口：

- 集市：承载二手、失物、求助、兼职、租房等校园生活信息。
- 树洞：承载匿名或弱身份表达场景，包括吐槽、倾诉、心愿和讨论类内容。
- 个人中心：承载登录状态、校园认证、我的发布、我的评论、我的喜欢、回复通知、反馈和其他个人功能。

## 功能概览

### 集市信息流

- 支持多分类内容浏览。
- 支持按校区筛选内容。
- 支持搜索入口。
- 支持信息详情页查看。
- 支持图片内容预览。
- 支持点赞、评论、价格或热度信息展示。
- 支持触底加载更多。
- 支持加载态、骨架屏和基础过渡动画。
- 支持左右滑动切换主分类，并在边界处使用阻尼回弹。

### 树洞信息流

- 支持树洞分类浏览。
- 支持新发、新回、热门、精选等排序维度。
- 支持热榜入口。
- 支持树洞风格卡片展示。
- 支持详情、评论、点赞和回复互动。
- 支持骨架屏、加载动画、列表过渡和滑动切换体验。

### 详情与评论

- 支持内容详情展示。
- 支持图片预览。
- 支持联系、返回、举报等操作入口。
- 支持评论列表、二级回复、点赞和删除。
- 支持评论排序切换。
- 支持评论区骨架屏和分页加载态。
- 支持评论区左右滑动切换排序，并避免影响点击、长按和图片预览。

### 内容发布

- 支持多类型内容发布。
- 支持表单输入、图片上传和发布状态保护。
- 支持本地上传中状态，不使用遮挡全屏的加载提示。
- 支持发布过程中的重复提交防护。

### 个人中心

- 支持登录和用户状态入口。
- 支持校园认证相关页面。
- 支持我的发布、我的评论、我的喜欢、我的回复。
- 支持客服、反馈、协议、申请等辅助入口。
- 支持轻量动画和按压反馈。

### 小工具

- 内置 2048 小游戏页面。
- 支持二维码相关页面。
- 支持 WebView 承载外部内容页面。

## 页面结构

主要页面包括：

- 首页：集市信息流、分类导航、校区筛选、搜索入口。
- 搜索页：内容搜索和搜索结果展示。
- 详情页：帖子详情、图片预览、评论和回复。
- 树洞页：树洞信息流、热榜入口、分类和排序。
- 热榜页：热门内容展示。
- 发布页：内容发布入口。
- 个人中心：首页式个人功能入口。
- 我的评论：用户评论记录。
- 我的喜欢：用户点赞或收藏记录。
- 我的发布：用户发布记录。
- 我的回复：收到的回复记录。
- 联系客服：客服和帮助入口。
- 意见反馈：建议和反馈入口。
- 登录页：用户登录相关流程。
- 认证申请页：校园认证相关流程。
- 协议页：用户协议或说明内容。
- 二维码页：二维码展示或分享相关功能。
- WebView 页：外部页面承载。
- 2048 小游戏：轻量娱乐功能。

## 组件与工具

项目中包含以下自定义组件和工具能力：

- 导航组件：统一处理页面导航视觉。
- 搜索栏组件：提供搜索输入和清空能力。
- 径向菜单组件：提供发布入口或快捷操作入口。
- 图片上传组件：统一处理图片选择、预览、删除、上传中状态。
- 上传服务工具：封装图片上传流程。
- 会话工具：封装认证请求头和登录态相关处理。
- API 兼容工具：统一处理后端错误响应。
- 内容检查工具：辅助内容安全检查。
- 滑动切换工具：提供 tab 左右滑动识别、边界阻尼和回弹动画。

1. 使用微信开发者工具导入本项目目录。

## 协作建议

- 修改页面前先确认是否影响 tabBar、导航栏、浮动按钮和滚动区域。
- 调整动效时只使用 transform 和 opacity，避免改变布局尺寸。
- 骨架屏应尽量复用真实卡片尺寸，避免加载态跳动。
- 上传、发布、评论等操作必须有提交中状态。
- 涉及用户内容的变更必须考虑内容审核和异常回退。
- 提交前建议至少完成页面预览、基础点击链路、触底加载、左右滑动和表单提交检查。

## 质量检查清单

提交前建议检查：

- 首页列表能正常加载、筛选、触底加载和进入详情。
- 树洞列表能正常切换分类、排序和进入详情。
- 详情页能正常展示内容、图片、评论和回复。
- 评论、点赞、删除、举报等入口不被动效影响。
- 发布页上传和提交状态正常。
- 个人中心各入口可访问。
- 暗色模式下文字和骨架屏可读。
- 左右滑动不会带动整页横向偏移。
- 首尾 tab 滑动只回弹，不触发越界请求。
- README、截图和日志不包含敏感信息。
